## 获取指南

### Quickdraw
1. 创建一个新的 Issue 或 Pull Request
2. 在 5 分钟内关闭它
3. 成就自动授予

### YOLO
1. 创建一个 Pull Request
2. 不经过代码审查直接合并
3. 成就自动授予

### Starstruck
1. 创建一个有价值的开源项目
2. 推广项目获得社区关注
3. 当 Stars 达到对应等级时自动解锁

## 常见问题

1. **成就不显示？** — 可能需要最多 24 小时才会显示
2. **私有仓库有效吗？** — 需要在设置中启用"Show private contributions & achievements"
3. **Fork 的仓库算吗？** — 在 Fork 中的工作不计入，必须向父仓库提交 PR
4. **如何关闭成就显示？** — Settings → Profile → 取消勾选"Show achievements"
