# GitHub 学习路径

> 系统化的学习路线，从入门到专家。

## 路径一：新手入门（推荐 1-2 周）

目标：掌握 GitHub 基础操作，能够参与开源项目。

```
Day 1-2: 了解平台
├── [GitHub 入门教程](../basics/getting-started.md)
├── [什么是 GitHub](https://docs.github.com/zh/get-started/about-github)
└── 创建你的第一个仓库

Day 3-4: Git 基础
├── [Git 基础操作](../basics/git-basics.md)
├── [GitHub Skills - Introduction to GitHub](https://skills.github.com/)
└── 练习：提交代码到 GitHub

Day 5-6: 仓库管理
├── [仓库管理指南](../basics/repository-management.md)
├── 配置 SSH 密钥
└── 学习 README 和 .gitignore

Day 7: Pull Request
├── [Pull Request 指南](../collaboration/pull-requests.md)
└── 练习：Fork 一个项目并提交 PR
```

## 路径二：协作开发（推荐 2-3 周）

目标：掌握团队协作流程和项目管理。

```
Week 1: Git 工作流
├── [Git 工作流](../basics/git-workflow.md)
├── 分支策略（Git Flow / GitHub Flow）
└── 练习：团队协作项目

Week 2: 协作工具
├── [Issues 管理](../collaboration/issues.md)
├── [代码审查](../collaboration/code-review.md)
├── Projects 看板
└── Milestones 里程碑

Week 3: 自动化入门
├── [首个 Actions 工作流](../actions-advanced/first-workflow.md)
├── CI/CD 基础
└── [GitHub Pages 快速部署](../github-pages/quickstart.md)
```

## 路径三：专业认证（推荐 1-2 月）

目标：通过 GitHub 官方认证，提升专业竞争力。

```
Phase 1: GitHub Foundations（2-3 周）
├── 完成所有基础教程
├── [GitHub Skills](https://skills.github.com/) 全部课程
├── 阅读官方文档
└── 报名并通过 GHFC 考试

Phase 2: Actions 认证（2-3 周）
├── [Actions 高级教程](../actions-advanced/)
├── 构建完整的 CI/CD 管道
└── 报名并通过 GHAC 考试

Phase 3: Security 认证（2-3 周）
├── [安全教程](../security/)
├── CodeQL 实践
└── 报名并通过 GHSC 考试
```

## 路径四：全栈 GitHub 专家（3 月+）

```
全部基础教程 → Actions 高级 → Security 高级 → Copilot → 全部认证
     ↓              ↓              ↓            ↓           ↓
  [基础](../basics/)  [Actions](../actions-advanced/) [安全](../security/) [Copilot](../copilot/) [认证](../certifications/)
```

## 学习检查清单

### 新手阶段
- [ ] 创建 GitHub 账号
- [ ] 创建第一个仓库
- [ ] 完成一次 commit/push
- [ ] 提交第一个 Pull Request
- [ ] 使用 GitHub Pages 部署网站

### 进阶阶段
- [ ] 掌握分支管理策略
- [ ] 使用 Actions 自动化工作流
- [ ] 配置 Dependabot 和 CodeQL
- [ ] 参与开源项目贡献
- [ ] 美化个人 Profile

### 专家阶段
- [ ] 获得至少 2 个 GitHub 认证
- [ ] 构建复杂的 Actions 工作流
- [ ] 管理团队和组织的 GitHub 环境
- [ ] 使用 Copilot 提升开发效率
- [ ] 发表 GitHub 最佳实践文章

---

> 选择适合你的路径开始学习吧！每个教程都提供了 MD 版本，方便阅读和收藏。
