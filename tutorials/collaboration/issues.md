# Issues 管理

参考来源：[GitHub Docs 入门](https://docs.github.com/zh/get-started)。

## Issue 用途

- 记录 Bug。
- 讨论新功能。
- 拆分任务。
- 关联 Pull Request。

## 推荐模板

```markdown
## 背景

## 复现步骤

## 期望结果

## 实际结果

## 环境信息
```

## 标签建议

| 标签 | 用途 |
| --- | --- |
| `bug` | 缺陷 |
| `enhancement` | 新功能 |
| `documentation` | 文档 |
| `good first issue` | 新手友好 |
