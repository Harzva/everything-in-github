# GitHub 教程体系

> 从入门到官方认证的系统化 GitHub 学习路径

## 学习路径总览

```
探索GitHub → 加入GitHub → 项目托管 → 协作工作流 → GitHub Pages → 扩展生态
    ↓            ↓            ↓            ↓              ↓            ↓
 了解平台      创建账号     创建仓库     Fork/Pull    个人主页      Gist/工具
 亮点与生态    浏览项目     分支管理     Request      项目主页      客户端
              建立社交     SSH/公钥     Review       域名/Jekyll   周边服务
                           Issues      Wiki
```

## 官方认证体系

GitHub 提供 5 种官方认证，展示你在 GitHub 技术与工作流方面的专业能力：

| 认证 | 图标 | 考试范围 | 难度 | 官方链接 |
|------|------|---------|------|---------|
| **GitHub Foundations** | 🎓 | 协作 / Git 产品 / Git 基础 / 仓库工作 | 入门 | [官方文档](https://docs.github.com/zh/get-started/showcase-your-expertise-with-github-certifications/about-github-certifications#github-foundations) |
| **GitHub Actions** | ⚡ | 简化工作流 / 自动化任务 / 优化软件管道 | 中级 | [官方文档](https://docs.github.com/zh/get-started/showcase-your-expertise-with-github-certifications/about-github-certifications#github-actions) |
| **GitHub Advanced Security** | 🛡️ | 漏洞识别 / 工作流安全性 / 安全实现 | 高级 | [官方文档](https://docs.github.com/zh/get-started/showcase-your-expertise-with-github-certifications/about-github-certifications#github-advanced-security) |
| **GitHub Administration** | ⚙️ | 仓库管理 / 工作流优化 / 高效协作 | 中级 | [官方文档](https://docs.github.com/zh/get-started/showcase-your-expertise-with-github-certifications/about-github-certifications) |
| **GitHub Copilot** | 🤖 | 负责任 AI / Copilot 功能与计划 | 中级 | [官方文档](https://docs.github.com/zh/get-started/showcase-your-expertise-with-github-certifications/about-github-certifications#github-copilot) |

## 学习路径推荐

### 路径一：新手入门（1-2周）
```
GitHub 入门教程 → Git 基础 → 仓库管理 → Pull Request → GitHub Pages
```

### 路径二：协作开发（2-3周）
```
Git 工作流 → Issues 管理 → 代码审查 → Actions 自动化 → Security 基础
```

### 路径三：专业认证（1-2月）
```
GitHub Foundations 认证 → Actions 认证 → Security 认证 → Administration 认证
```

### 路径四：全栈 GitHub（3月+）
```
所有基础教程 → Actions 高级 → Security 高级 → Copilot → 全部认证
```
