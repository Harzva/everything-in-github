## 这是什么？

**Everything in GitHub** 是你的 GitHub 知识地图 — 一个全面收集 GitHub 生态资源、教程和工具的综合仓库。

基于 GitHub 官方 Certifications 文档与《GitHub 使用指南》整理，覆盖从新手入门到官方认证的完整学习路径。

### 核心内容

| 板块 | 内容 | 数量 |
|------|------|------|
| **官方认证体系** | 5 种 GitHub 官方认证详解 | 5 个认证 |
| **使用教程** | 从入门到进阶的系统化教程 | 20+ 篇 |
| **学习路径** | 4 条推荐学习路线 | 4 条路径 |
| **主题资源** | 成就徽章、Profile 工具、Actions 等 | 70+ 仓库 |
| **外部精选** | 社区优质教程和官方文档链接 | 10+ 链接 |

## GitHub Pages 网站

本项目包含一个精美的 GitHub Pages 网站，提供：

- **主题分类导航** — 5 大主题，结构化浏览
- **教程体系** — 官方认证 + 使用教程 + 学习路径
- **搜索/筛选功能** — 按类别、Stars、语言筛选
- **徽章/图标预览** — 可视化展示所有工具效果
- **MD版本切换** — 一键切换到 Markdown 阅读模式
- **响应式设计** — 完美适配桌面和移动设备
- **暗色科技主题** — 沉浸式的浏览体验
